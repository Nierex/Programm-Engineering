## Лабораторная работа №2
Класс итератор с гибкой настройкой и удобными применениями.
Оформленный код
class CountDown:
    def __init__(self, start):
        self.count = start + 1
    def __iter__(self):
        return self
    def __next__(self):
        self.count -= 1
        if self.count < 0:
            raise StopIteration
        return self.count

if __name__ == '__main__':
    counter = CountDown(5)
    for i in counter:
        print(i)

![Тема_11](https://github.com/Nierex/Programm-Engineering/blob/main/pic/Lab11_1.2.png)

- Вывод
Создан пользовательский итератор через класс с методами __iter__ и __next__. Это даёт полный контроль над логикой итерации — здесь реализован обратный отсчёт. Можно переиспользовать, настраивать начальное значение.

## Лабораторная работа №5
Такой же счетчик, как и в первом задании, только это генератор и использует yield.
Оформленный код
def countdown(count):
    while count >= 0:
        yield count
        count -= 1

if __name__ == '__main__':
    counter = countdown(5)
    for i in counter:
        print(i)
![Тема_11](https://github.com/Nierex/Programm-Engineering/blob/main/pic/Lab11_1.5.png)

- Вывод
Функция-генератор с yield — это компактный и мощный способ создавать итераторы. Она «замораживается» после каждого yield и возобновляется при следующем вызове. Идеально подходит для бесконечных или больших последовательностей, где не нужно хранить всё в памяти.

## Самостоятельная работа №1
Вас никак не могут оставить числа Фибоначчи, очень уж они вас заинтересовали. Изучив новые возможности Python, вы решили реализовать программу, которая считает числа Фибоначчи при помощи итераторов. Расчёт начинается с чисел 1 и 1. Создайте функцию fib(n), генерирующую n чисел Фибоначчи с минимальными затратами ресурсов. Для реализации этой функции потребуется обратиться к инструкции yield (она не сохраняет в оперативной памяти огромную последовательность, а даёт возможность “доставать” промежуточные результаты по одному). Результатом решения задачи будет листинг кода и вывод в консоль с числом Фибоначчи от 200.
- Оформленный код
  def fib(n):
    a, b = 1, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

for num in fib(200):
    print(num)
![Тема_11](https://github.com/Nierex/Programm-Engineering/blob/main/pic/Lab11_2.1.png)
- Вывод
  Использование yield позволяет создать генератор, который вычисляет числа Фибоначчи "на лету", не храня всю последовательность в памяти. Это экономит ресурсы и подходит для больших значений n. Код выводит все 200 чисел в консоль — это демонстрация работы генератора.
## Самостоятельная работа №2
К коду предыдущей задачи добавьте запоминание каждого числа Фибоначчи в файл “fbl.txt”, при этом каждое число должно находиться на отдельной строчке. Результатом выполнения задачи будет листинг кода и скриншот получившегося файла.
Оформленный код
def fib(n):
    a, b = 1, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

with open('fbl.txt', 'w') as file:
    for num in fib(200):
        file.write(str(num) + '\n')
